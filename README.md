# Going Merry

從新手村啟航時的基本配備, 存在 Git 上方便隨時逃難!?

# bashrc

- 複製設定檔 `.bashrc` 到家目錄下
- 如果重新連線沒有自動載入, 改名: `.profile`

# tmux

- 設定檔: `.tmux.conf`
- 複製到家目錄下, `tmux source-file ~/.tmux.conf` 或者重新連線會自動載入這個版本, 之後都可以用 `r` 重載
- command prefix: `ctrl + a`
- split: `v` & `b`
- 在panes間切換: `h` `j` `k` `l`
- 其他差別不多, 直接看設定檔
